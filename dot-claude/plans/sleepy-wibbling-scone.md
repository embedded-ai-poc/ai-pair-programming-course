# 피카츄 배구 게임 구현 계획

## 개요
HTML5 Canvas + 순수 JavaScript로 피카츄 배구 스타일의 1인용 배구 게임 구현

## 게임 사양
- **모드**: 1인용 (플레이어 vs AI)
- **승리 조건**: 15점 선취
- **캐릭터**: Canvas로 직접 그리는 간단한 캐릭터 (눈, 귀 포함)
- **조작**: 방향키(이동/점프) + 스페이스바(공격)

## 구현 단계

### 1단계: 기본 세팅
- [ ] `index.html` - Canvas 요소, 기본 구조
- [ ] `style.css` - 캔버스 중앙 정렬, 배경
- [ ] `main.js` - Canvas 초기화, 게임 루프 (`requestAnimationFrame`)

### 2단계: 플레이어 구현
- [ ] `player.js` - Player 클래스
  - 위치, 속도, 크기 속성
  - 좌우 이동, 점프 메서드
  - 중력 적용
  - 바닥 충돌 처리

### 3단계: 키보드 입력
- [ ] `main.js`에 키보드 이벤트 리스너
  - 방향키: 좌우 이동, 위 점프
  - 스페이스바: 공격 (스파이크)

### 4단계: 공 구현
- [ ] `ball.js` - Ball 클래스
  - 위치, 속도, 반지름 속성
  - 중력, 공기 저항 적용
  - 벽/바닥/네트 충돌 처리

### 5단계: 물리/충돌
- [ ] `physics.js`
  - 원-원 충돌 감지 (플레이어-공)
  - 충돌 시 공 반사 계산
  - 네트 충돌 처리

### 6단계: 렌더링
- [ ] `renderer.js`
  - 배경 (하늘, 바닥)
  - 네트 그리기
  - 캐릭터 그리기 (몸통, 눈, 귀, 볼)
  - 공 그리기
  - 점수 표시

### 7단계: 점수 시스템
- [ ] 공이 바닥에 닿으면 득점 판정
- [ ] 15점 도달 시 게임 종료
- [ ] 승리/패배 화면

### 8단계: AI 구현
- [ ] `ai.js` - AI 클래스
  - 공 위치 추적
  - 착지 지점 예측
  - 이동 및 점프 결정
  - 난이도 조절 (반응 속도)

### 9단계: 게임 흐름
- [ ] 시작 화면
- [ ] 라운드 시작/리셋
- [ ] 게임 오버 화면
- [ ] 재시작 기능
