# Deep Research 시작

deep-research 스킬을 실행합니다.

**Arguments**: $ARGUMENTS

사용자가 전달한 조사 주제: `$ARGUMENTS`

이 명령은 deep-research 스킬의 7-Phase 파이프라인을 시작합니다:
1. SCOPE → 2. PLAN → 3. SEARCH (병렬) → 4. VALIDATE → 5. ANALYZE → 6. SYNTHESIZE → 7. REPORT

## 실행

deep-research 스킬의 SKILL.md를 읽고 Phase 1: SCOPE부터 시작하세요.

**참고**: 조사 범위에 따라 Quick(3-phase) 또는 Standard/Deep(5-7 phase) 모드를 선택하세요.
